Enhancement Two



  • Enhancement Two: Algorithms and Data Structure

    During my previous code review[Watch The Video], I listed some of the weaknesses that were caused by usage of unsuitable data structures and algorithms. Firstly, the application freezes once an admin logs in, this is due to the large amount of data from fetched from database at start up and loaded at once on the Main GUI. Also there is not suitable algorithm implemented in the project to make searching of forms, student’s details. The project therefore depends on the select statements to search for the required data.

    To enhance this project, I corrected the first issue, freezing of the application at start up. To make it work, I created an array List which are used to hold the data fetched from the database. This data is further user to create only ten of the forms which need to be displayed in the main window. This forms are stored in a VBox layout, a scroll pane is used to hold this layout to allow for scrolling so that a user can see all the forms. At the end of the first ten, I add a button, this button add ten more forms of each data in the array list. Since only ten records are loaded at start up, the freezing of the application is removed.

    The second issue was about the searching for student’s details, student’s examination report forms and other necessary details like fee records. This part of the project is one of the reason for the development of the project. Fetching and display of data should be quick and easy. On the old application, user can search for the data, however due to some weak algorithms used, the data collected and the time taken to display the data is not meeting the objectives of the application. To correct this problem I introduced a searching algorithm which fetches the necessary data from the database, filter the data and display it to the user.

    The enhancements listed in my previous code review have been met and the program is now improved. Searching of data is quick and easy. The application does not freeze at admin’s log in. However I am still checking the best algorithm to use so that all users, regardless of the memory capacity of their computers, should have the application working and meeting its objectives.

  • Achievement

    These two enhancements on data structures and algorithms has enabled the full project to meet fully the objectives of the application. The project takes few seconds to load unlike the old application. Secondly, the deep research on the data structures and algorithms perfected me on this field. During the research I worked on most of the java data structures like queues, stacks, Binary Trees, Binary search trees and many more. I was able to check algorithms time complexity, efficiency and choosing of algorithms depending on memory usage. However I am still working on the algorithm part, to make sure I get the best algorithms depending on the application usage and the targeted computer systems.
