    Software Design/Engineering

    After my code review classes, I found out that, beside the application running without syntax errors and or logical errors, there are some other programming designs, coding rules , data structures and algorithms etc. that need to be enhanced further to make the project codes more readable, easy to debug and also make the project more powerful and reliable.

    For the software design and engineering part of the project, I found a lot that needs to be enhanced. The project lacks enough documentation which makes it difficult to debug and read, secondly, the project has incomplete IF-ELSE statements and try-catch blocks without exception messages. Furthermore, the application failed to follow the basic java standards rules of coding. There to enhance the application, I will use my knowledge from computer science program and other previous knowledge to correct and correct all the selected areas above.

  • Algorithms and Data Structures

    For the algorithms and data structures portion of the project, I found out several weaknesses in the code that actually caused the application to freeze during run time, which makes the application to take a lot of time at start up and also very times one opens the activity window. From my view, I checked and confirmed that some algorithms and data structures which are not suitable had been used.

    Therefore to enhance the application I used my knowledge and some Internet resources to obtain the most suitable data structures and algorithms to use. This included searching algorithms, use of array list and other java data structures.

  • Databases

    For the databases portion of the final project, there were some changes which needs to be done, instead of using a file based database, SQLite, I propose to use a web based database so that the app can support data sharing. Further it will secure the data and also back up of students data will be carried out easily.

    To achieve the above task I selected the best SQL Database hosting services to create and host my database for the application. The most challenging part at this point was to get a hosting site which is reliable, affordable and also support all the services for connecting the database remotely.
